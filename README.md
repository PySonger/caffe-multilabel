# caffe-multilabel #
## Description ##
This repo contains these files:
- deploy.prototxt

shows an example about how to do multi-label classification with caffe.

- data_multilabel.py

shows how to use python layer to feed data to network.

- test_multilabel.py

shows how to test/evalute trained caffemodel.

## Requirement ##
- caffe & pycaffe
- python = 2.7
